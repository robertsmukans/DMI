# - veikt komentaru nano logâ
$ echo – ar so komandu var ievadit tekstu, kas tiks attelots ekrana
$PATH – pievienojam izveleto direktoriju celiem
Cd .. – var iziet no mapes
Cd ~ - aiziet atpakal uz home directory
Cd skola – var aiziet uz mapi skola
Chmod 744 – veikt korekcijas sajâ faila
Chromium & - atver chrome pârlûkprogrammu
Cp mape1 mape2 – mapes 1 saturu parkope uz mape2
Echo $PATH – redzam caur kâdiem celiem atrodas izvçleta direktorija
Firefox & - atver firefox parlûkprogrammu
Git add . – pievieno kontu
Git commit –m “datumslaiks” –  iestata faila datumu un laiku
Git config –global user.email – piesaista savam github kontam
Git push origin master -  augsupielade datus no terminala github kontâ
History – parada rakstito vesturi
History > hystory1 – vesturi parnes uz jaunu failu
Ls – parâda eksistçjosas mapes
Ls –a – parâda visus eksistejosos failus
Ls –l – parâda informaciju par failu
Ls –la – parâda veiktas darbibas uz esosas masinas
Mkdir – var izveidot jaunu mapi
Nano – atver logu, kur var veidot skriptu
Pwd – parâda kur sobrîd atrodos
Readonly name – saglaba so mainîgo nemainigu visu darbibas laiku
Rmdir – var dzçst kâdu mapi
Uname –r – parada sistemas versiju
Who – parada uz kâ es strâdâju
Whoami – parada kas es esmu masina
